Musik konfigurieren:
TypeScript

Copy
window.MusicKit.configure({
  developerToken: process.env.NEXT_PUBLIC_MK_DEV_TOKEN!,
  app: { name: 'GlassBeats', build: '1.0.0' }
});
Auth: Button “Mit Apple anmelden” → MusicKit.authorize() → User-Token in http-only Cookie speichern.
Playback: MusicKit.getInstance().player.play({ type: 'songs', id: '...' }) – Gapless, Crossfade, Lossless, Atmos automatisch via Apple.
Cloud-Library: Upload-Endpunkt /v1/me/library/uploads – UI-Button “Datei hochladen” → presigned URL → PUT → erscheint in Bibliothek.
Radio-Stationen: createStation({ type: 'artists', id: '...' }) → endloser Feed.
Lyrics: /v1/catalog/de/songs/{id}/lyrics → timed array → full-screen Overlay mit Wort-für-Wort-Highlight.
